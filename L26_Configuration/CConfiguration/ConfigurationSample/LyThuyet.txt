---------------- Configuration -------------------- chứa các thông tin cấu hình cho chương trình-> viết một chương trình triển khai trên nhiều nền tảng khác nhau, ta cần có các thông tin khác nhauthay vì code, triển khai lại và sửa đổi thông tin đó thì ta đưa cấu hình đó ra môi trường bên ngoàivà thay đổi nó mà không phải cài đặt lại chương trìnhtrong .net, cung cấp 2 cấp độ Configuration - cấp độ thứ nhất: là các configuration qua lớp trừu tượng, cung cấp một nền tảng thông tin thống nhất để truy cập vào các thiết lập cấu hình.- cấp độ thứ hai: là configuration provider, làm việc trực tiếp với một dạng thiết lập cấu hình nào đó (đọc qua configuration abstraction nào đó của nó)- cung cấp nhiều loại hỗ trợ để thiết lập configuration -> chủ yếu là dùng file setting: json, xml, ini-> biến môi trường -> command line-> custorm provider-> directory file-> inmemory .net-> thread party
